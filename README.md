# Kaz Clothing Shop

## Description
An online store for a Kazakh clothing brand. The project uses Django Templates for the frontend and Django REST Framework for the backend.

## Tech Stack
- Django, Django Templates
- Django REST Framework
- SQLite
- django-cors-headers

## How to run

```bash
git clone <this-repo-link>
cd kaz-clothing-shop
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
